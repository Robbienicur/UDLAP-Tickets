# UDLAP Tickets

Aplicacion movil para la venta de boletos digitales para el estacionamiento de la Universidad de las Americas Puebla (UDLAP).

## Descripcion

UDLAP Tickets es una app desarrollada en Flutter que permite a los estudiantes y personal de la UDLAP comprar y gestionar boletos digitales para acceder al estacionamiento universitario de manera rapida y sencilla, eliminando la necesidad de boletos fisicos.

## Funcionalidades principales

- Registro e inicio de sesion de usuarios
- Compra de boletos de estacionamiento
- Pago con tarjeta o saldo recargable
- Visualizacion de boletos activos
- Historial de compras
- Perfil de usuario

## Tecnologias

- Flutter / Dart
- Material Design

## Instalacion

1. Clonar el repositorio
```bash
git clone https://github.com/Robbienicur/UDLAP-Tickets.git
```

2. Instalar dependencias
```bash
flutter pub get
```

3. Ejecutar la aplicacion
```bash
flutter run
```
